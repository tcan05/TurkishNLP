# TurkishNLP

## About This Project
This is a syllable-based NLP project. It first creates syllables from a corpus and then uses probability-based approach to create sentences and calculates perplexity.

## Requirements
- Python 3.10+
- collections

Install Python dependencies:
```bash
pip install collections
```

## Installation
1. Clone the repository:
```bash
git clone https://github.com/tcan05/TurkishNLP.git
```
2. Navigate to the project directory:
```bash
cd TurkishNLP/Project
```
3. Run the application:
```bash
python Test.py
```
